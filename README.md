# Maccordion

맥북의 힌지 각도를 "벨로우즈"처럼 쓰는 간단한 macOS 아코디언 프로토타입입니다.

## 어떻게 동작하나요?

- 맥북의 비공개 `Lid Angle Sensor` 값을 `IOKit` HID feature report로 읽습니다.
- `A S D F G H J K L` 키가 음정을 담당합니다.
- 화면을 여닫는 속도가 벨로우즈 압력처럼 작동해 볼륨이 달라집니다.
- 화면을 여는 중인지 닫는 중인지에 따라 음색도 조금 바뀝니다.

## 요구 사항

- macOS 14 이상
- Swift 6 toolchain
- 힌지 센서가 있는 MacBook
  - 보통 2019년 이후 모델에서 기대할 수 있습니다.
  - 공개 API가 아니므로 일부 M1/M2 모델에서는 동작하지 않을 수 있습니다.

## 실행

```bash
swift run
```

창이 뜨면:

1. `A S D F G H J K L` 중 하나를 누릅니다.
2. 키를 누른 채로 디스플레이를 조금씩 여닫습니다.
3. 힌지 움직임이 압력처럼 반영되며 소리가 납니다.

## 주의

- 이 코드는 Apple의 공개 센서 API를 쓰지 않습니다.
- 기기마다 센서 허브 구성이 달라 동작하지 않을 수 있습니다.
- 힌지를 너무 세게 흔들면 하드웨어에 좋지 않으니 천천히 테스트하는 편이 안전합니다.
